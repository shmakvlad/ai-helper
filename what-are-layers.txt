  ЧТО ТАКОЕ СЛОИ В МОДЕЛИ
══════════════════════════════════════════════════════════════════════════════════
Коротко: слой — это один «этаж» обработки текста. На каждом этаже одно и то же оборудование (Attention + FFN),
но разные настроенные параметры. Текст проходит через все 32 этажа последовательно — от простого распознавания слов на 1-м этаже
до формирования ответа на 32-м. Каждый следующий слой строит понимание поверх того, что вычислил предыдущий

  ╔═══════════════════════════════════════════════════════════════════════╗
  ║  АНАЛОГИЯ: Этажи фабрики                                           ║
  ║                                                                     ║
  ║  Представь фабрику с 32 этажами. На каждом этаже — одинаковое      ║
  ║  оборудование: конвейер + лаборатория. Но рабочие (параметры)      ║
  ║  на каждом этаже РАЗНЫЕ и обучены по-своему.                        ║
  ║                                                                     ║
  ║  Сырьё (текст) заезжает на 1-й этаж, обрабатывается,               ║
  ║  поднимается на 2-й, снова обрабатывается, и так до 32-го.        ║
  ║  С каждым этажом изделие (понимание текста) становится              ║
  ║  всё более готовым.                                                 ║
  ╚═══════════════════════════════════════════════════════════════════════╝


  СЛОЙ — это один «этаж» обработки. Каждый слой содержит:

  ┌────────────────────────────────────────────────────────────────┐
  │                                                                │
  │  ОДИН СЛОЙ = Attention + FFN                                   │
  │                                                                │
  │  1) Multi-Head Attention  — токены общаются друг с другом     │
  │     (W_Q, W_K, W_V, W_O — свои для каждого слоя)             │
  │                                                                │
  │  2) Feed-Forward Network  — каждый токен думает сам            │
  │     (W₁, W₂ — свои для каждого слоя)                          │
  │                                                                │
  │  Параметров на слой: ~201 000 000                              │
  │                                                                │
  └────────────────────────────────────────────────────────────────┘


  Все слои имеют одинаковую СТРУКТУРУ, но РАЗНЫЕ параметры:

  ┌────────────────────────────────────────────────────────────────┐
  │                                                                │
  │  Вход: "Столица Франции —"  (матрица 5 × 4096)               │
  │    │                                                           │
  │    ▼                                                           │
  │  ┌──────────┐                                                  │
  │  │ Слой 1   │  Распознаёт части речи, базовые связи           │
  │  │ 201M     │  "Фран" + "ции" = одно слово                    │
  │  └────┬─────┘                                                  │
  │       ▼                                                        │
  │  ┌──────────┐                                                  │
  │  │ Слой 2   │  Грамматические роли                             │
  │  │ 201M     │  "Столица" — подлежащее                          │
  │  └────┬─────┘                                                  │
  │       ▼                                                        │
  │      ...                                                       │
  │       ▼                                                        │
  │  ┌──────────┐                                                  │
  │  │ Слой 10  │  "Столица Франции" — географический вопрос       │
  │  │ 201M     │                                                  │
  │  └────┬─────┘                                                  │
  │       ▼                                                        │
  │      ...                                                       │
  │       ▼                                                        │
  │  ┌──────────┐                                                  │
  │  │ Слой 25  │  Франция → столица → Париж                       │
  │  │ 201M     │                                                  │
  │  └────┬─────┘                                                  │
  │       ▼                                                        │
  │      ...                                                       │
  │       ▼                                                        │
  │  ┌──────────┐                                                  │
  │  │ Слой 32  │  Формирует финальный ответ: "Париж"              │
  │  │ 201M     │                                                  │
  │  └────┬─────┘                                                  │
  │       ▼                                                        │
  │  Выход: (матрица 5 × 4096, но теперь с "пониманием")          │
  │                                                                │
  └────────────────────────────────────────────────────────────────┘

  Размерность НЕ МЕНЯЕТСЯ: на входе 5×4096, на выходе 5×4096.
  Меняется только содержание — от «сырых слов» к «пониманию».


  ЗАЧЕМ МНОГО СЛОЁВ, А НЕ ОДИН БОЛЬШОЙ?
  ───────────────────────────────────────

  ╔═══════════════════════════════════════════════════════════════════════╗
  ║  АНАЛОГИЯ: Почему нельзя выучить алгебру за один урок               ║
  ║                                                                     ║
  ║  Урок 1: цифры                                                      ║
  ║  Урок 2: сложение (нужны цифры из урока 1)                         ║
  ║  Урок 3: умножение (нужно сложение из урока 2)                     ║
  ║  Урок 4: уравнения (нужно умножение из урока 3)                    ║
  ║                                                                     ║
  ║  Каждый следующий урок опирается на предыдущий.                    ║
  ║  Нельзя перепрыгнуть — нужно идти последовательно.                 ║
  ║  Слои работают так же: каждый строит на результатах предыдущего.   ║
  ╚═══════════════════════════════════════════════════════════════════════╝

  Слой 1 не может сразу понять "нужен Париж" — он ещё не знает,
  что "Фран" + "ции" = "Франции". Слой 10 не может спланировать
  ответ — он ещё не понял, что вопрос о столице. Каждый слой
  добавляет свой уровень понимания поверх предыдущего.


  СКОЛЬКО СЛОЁВ В РАЗНЫХ МОДЕЛЯХ:
  ────────────────────────────────

  ┌─────────────────┬───────────┬──────────────────┐
  │ Модель          │ Слоёв     │ Параметров всего │
  ├─────────────────┼───────────┼──────────────────┤
  │ GPT-2 Small     │    12     │     0.1B         │
  │ LLaMA 7B        │    32     │       7B         │
  │ LLaMA 70B       │    80     │      70B         │
  │ GPT-3           │    96     │     175B         │
  │ GPT-4 (оценка)  │   120+    │    1000B+        │
  └─────────────────┴───────────┴──────────────────┘

  Больше слоёв = глубже «мышление» = лучше качество.
  Но и дороже, медленнее, больше памяти.
